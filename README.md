# graphql-type-json [![Travis][build-badge]][build] [![npm][npm-badge]][npm]

JSON scalar type for [GraphQL.js](https://github.com/graphql/graphql-js).

[![Codecov][codecov-badge]][codecov]

## Usage

```js
import GraphQLJSON from 'graphql-type-json';
```
